# Security Policy

## Supported Versions

We provide security updates for the latest minor release line.

| Version | Supported |
| ------- | --------- |
| 1.x     | :x:       |
| 0.3.x   | :white_check_mark: |
| 0.2.x   | :white_check_mark: |
| 0.1.x   | :x:       |

## Reporting a Vulnerability

Please report security issues privately.

- Email: ticketarcade.official@gmail.com
- Subject: "Security Disclosure - Ticket Arcade"

What to include:
- A clear description of the issue and impact
- Steps to reproduce or a proof of concept
- Any suggested fixes or mitigations

What to expect:
- We aim to acknowledge reports within 72 hours.
- We will provide a status update within 7 days.
- If accepted, we will coordinate a fix and disclosure timeline with you.

Please do not open public issues or discussions for security vulnerabilities.
